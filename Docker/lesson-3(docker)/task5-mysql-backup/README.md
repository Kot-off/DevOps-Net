#  ЗАДАНИЕ 5: Резервное копирование MySQL (*)

## Цель задания
1. Написать bash-скрипт для резервного копирования БД MySQL
2. Использовать образ `schnitzler/mysqldump` через `docker run`
3. Настроить выполнение каждую минуту через cron
4. Сохранять бэкапы в `/opt/backup`
5. Не показывать логин/пароль в git

## Требования
- Скрипт должен запускаться в сети "backend"
- Использовать команду `docker run` с образом `schnitzler/mysqldump`
- Хранить последние 10 бэкапов
- Пароли должны быть защищены (не в коде скрипта)
- Настроить cron на выполнение каждую минуту

## Файлы в папке
- `backup-mysql.sh` - скрипт резервного копирования
- `README.md` - эта инструкция

## Команды
```bash
# Делаем скрипт исполняемым
chmod +x backup-mysql.sh

# Проверяем синтаксис
bash -n backup-mysql.sh

# Создаем директорию для бэкапов
sudo mkdir -p /opt/backup
sudo chown $USER:$USER /opt/backup

# Запускаем скрипт
./backup-mysql.sh

# Проверяем результат
ls -la /opt/backup/
```
